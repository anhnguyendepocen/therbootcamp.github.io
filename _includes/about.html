R Courses for beginners, academics, and professionals. We teach basic R, frequentist and Bayesian statistics, and data visualization. We introduce you to R's highly efficient tidyverse library and documentation tools for reproducible reports. We demonstrate big data applications and cutting-edge machine-learning techniques.

Basics


Getting started with R and RStudio. Installing software and packages. Reading data from external files and doing basic manipulations.

Statistics


Descriptive and inferential statistics. Hypothesis testing with t-tests, ANOVA and regression. Machine learning and statistical prediction with decision trees, random forests and support vector machines.

Visualisation


Histograms, barplots and scatterplots. Networks and maps. Interactive graphics with Shiny.

Productivity


Debugging. Writing fast code with C++ and parallel programming. Writing, organizing, and sharing reproducible code using R Markdown and GitHub.
