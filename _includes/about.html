The R Course


An in-depth course tailored to beginners. For academics, and professionals. We cover classic frequentist and modern Bayesian statistics. We show you how to make elegant, informative data visualizations. We introduce you to R's highly efficient tidyverse library and documentation tools for reproducible reports. We demonstrate big data applications and cutting-edge machine-learning techniques.

Essentials


Getting started with R and RStudio. Installing software and packages. Getting your data into and out of R and doing basic manipulations.

Statistics


Descriptive and inferential statistics. Hypothesis testing with t-tests and ANOVA. Machine learning and statistical prediction with regression, decision trees, and support vector machines.

Visualisation


Histograms, barplots, scatterplots (even our own pirateplots), networks and maps. Interactive graphics with Shiny.

Productivity


Writing, organizing, and sharing reproducible code using R Markdown and GitHub. Writing fast code with C++ and parallel programming.
